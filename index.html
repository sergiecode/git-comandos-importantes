Comandos importantes de Git

A continuación, se presentan algunos de los comandos más utilizados en Git:

Configuración

Configurar el nombre del autor:

git config --global user.name "Sergie Code" Tu Nombre

Configurar el correo electrónico del autor:

git config --global user.email "sergiecode@gmail.com" (Tu Mail)

Creación de repositorio

Iniciar un nuevo repositorio:

git init

Clonar un repositorio existente:

git clone nombre_de_usuario@servidor:path/al/repo.git

Registro de cambios

Ver el estado de los archivos:

git status

Agregar cambios al área de preparación:

git add nombre_archivo

Confirmar los cambios:

git commit -m "Mensaje del commit"

Ramas

Listar ramas:

git branch

Crear una nueva rama:

git branch nombre_rama

Cambiar a una rama específica:

git checkout nombre_rama

Crear y cambiar a una nueva rama:

git checkout -b nombre_rama

Fusión y Rebase

Fusionar una rama en la rama actual:

git merge nombre_rama

Rebase de una rama sobre la rama actual:

git rebase nombre_rama

Colaboración

Ver los repositorios remotos configurados:

git remote -v

Enviar cambios al repositorio remoto:

git push nombre_remoto nombre_rama

Obtener cambios del repositorio remoto:

git pull nombre_remoto nombre_rama

Historial

Ver el historial de commits:

git log

Ver cambios específicos entre commits:

git diff commit1 commit2
